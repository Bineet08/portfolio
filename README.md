## рџљЂ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Bineet08/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   
   Navigate to `http://localhost:5173` to view the portfolio.
